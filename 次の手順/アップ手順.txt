ステップ1：Firebaseプロジェクト作成

Firebaseコンソールにアクセス

「プロジェクトを追加」をクリック

プロジェクト名を入力（例：StarbucksVisited）

無料プラン（Sparkプラン）でOK

「Google Analytics」はオフで構わない

ステップ2：Authentication（認証）設定

コンソール左メニュー → Authentication → ログイン方法

「Google」を有効化

家族のGoogleアカウントだけがログインできるように許可

メールアドレスの制限ルールを設定可能（後で簡単に追加/削除）

ステップ3：Firestore（データベース）作成

コンソール左メニュー → Firestore Database → データベース作成

「テストモード」で作成（30日間全員アクセス可）

コレクション名：stores

ドキュメント例：

{
  "id": "001",
  "name": "渋谷店",
  "prefecture": "東京",
  "city": "渋谷区",
  "streetAddress": "神南1-23-3",
  "building": "渋谷ビル",
  "zipcode": "1500001",
  "visited": false,
  "favorite": false,
  "closed": false
}


すべての店舗をアップロード（JSONからバルクで入れられます）

ステップ4：ReactアプリにFirebase SDK導入

プロジェクトでFirebaseをインストール

npm install firebase


.env にFirebase設定情報を追加（APIキーなど）

REACT_APP_FIREBASE_API_KEY=xxxx
REACT_APP_FIREBASE_AUTH_DOMAIN=xxxx.firebaseapp.com
REACT_APP_FIREBASE_PROJECT_ID=xxxx
REACT_APP_FIREBASE_STORAGE_BUCKET=xxxx.appspot.com
REACT_APP_FIREBASE_MESSAGING_SENDER_ID=xxxx
REACT_APP_FIREBASE_APP_ID=xxxx


Firebase初期化（例：firebase.js）

import { initializeApp } from "firebase/app";
import { getFirestore } from "firebase/firestore";
import { getAuth, GoogleAuthProvider } from "firebase/auth";

const firebaseConfig = {
  apiKey: process.env.REACT_APP_FIREBASE_API_KEY,
  authDomain: process.env.REACT_APP_FIREBASE_AUTH_DOMAIN,
  projectId: process.env.REACT_APP_FIREBASE_PROJECT_ID,
  storageBucket: process.env.REACT_APP_FIREBASE_STORAGE_BUCKET,
  messagingSenderId: process.env.REACT_APP_FIREBASE_MESSAGING_SENDER_ID,
  appId: process.env.REACT_APP_FIREBASE_APP_ID,
};

const app = initializeApp(firebaseConfig);
export const db = getFirestore(app);
export const auth = getAuth(app);
export const provider = new GoogleAuthProvider();

ステップ5：Reactアプリで認証・Firestore連携

ログイン機能：Googleアカウントでログイン

Firestoreデータ取得：stores コレクションを読み込む

データ更新：訪問済み・お気に入り・追加・削除は update(), add(), delete() を使用

リアルタイム同期：onSnapshot を使うと家族全員が即時に最新データを閲覧可

ステップ6：Netlifyで公開

Reactアプリをビルド

npm run build


Netlifyにログイン → 新しいサイトを追加 → Deploy manually

/build フォルダをドラッグ＆ドロップ

公開URLが発行される

Firebase認証でログインしてからアプリを操作
